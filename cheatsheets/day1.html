⚡ Day 1 — Terminal & Git Cheat Sheet

7-Day AI Bootcamp · MentorBot · Print-friendly A4

📂 Navigation Commands

pwd
Print working directory
ls [-la]
List files (-l=long, -a=hidden)
cd ~/Desktop
Change directory
cd ..
Go up one level
mkdir my-project
Create new directory
rm -rf dir/
Remove directory (careful!)

📄 File Operations

touch hello.txt
Create empty file
cat file.txt
View file contents
nano file.txt
Edit file in terminal
cp src dest
Copy file
mv old new
Move/rename file
head/tail -n 10 file
First/Last 10 lines

🔄 Git Essentials

git init # Initialize repo git add . # Stage all changes git commit -m "msg" # Commit with message git push origin main # Push to GitHub git pull origin main # Pull latest changes git status # Check status git log --oneline # View commit history

🌐 API & Network

curl -s URL
Fetch URL content
curl -s URL | jq
Pretty-print JSON
curl -X POST -d '{}' URL
POST request
wget URL
Download file
# Example: Call public API curl -s https://api.github.com/users/octocat | jq '.login,.public_repos'

💡 Pro Tips

  • Use Tab for auto-complete in terminal
  • Ctrl+C to cancel running command
  • Ctrl+R to search command history
  • ↑ ↓ arrows to browse previous commands
  • Use .gitignore to exclude files from Git
  • Always pull before push to avoid conflicts
