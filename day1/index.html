Block 0 — CLI Ritual (every day)⏱ 30 min
🖥️ Daily Setup with Claude Code
Set up your workspace the way a professional AI Director does — every single day.
  1. Install Claude Code CLI (if not already): npm install -g @anthropic-ai/claude-code
  2. Create a project folder: mkdir ai-director && cd ai-director
  3. Run claude to start an interactive session
  4. Run /init — Claude analyzes your project and generates CLAUDE.md
  5. Review CLAUDE.md: add your name, project goal, preferred language
  6. Run /plan to practice planning mode before any code
  7. End of day ritual: /compact → commit your work
Resources: Official Claude Code Documentation Slash Commands (Interactive)
Practice
Block 1 — Reading⏱ 2 hrs
📚 CLI & Git Fundamentals
Build your foundation. Read, watch, and understand the core tools.
  1. Terminal basics (45 min): Navigation (cd, ls, pwd), file operations (touch, mkdir, cp, mv, rm), viewing files (cat, less, head, tail), piping (|, grep, sort)
  2. Git essentials (45 min): What is version control, clone/add/commit/push/pull, branches (create, switch, merge), resolving merge conflicts, .gitignore
  3. VS Code setup (15 min): Open project (code .), integrated terminal (Ctrl+`), command palette (Ctrl+Shift+P), essential extensions (GitLens, Thunder Client)
  4. Claude Code deep dive (15 min): /plan, /compact, /clear, /init, CLAUDE.md structure, context management
Resources: Claude Code Memory & CLAUDE.md Claude Code Cheat Sheet Pro Git Book (free) MDN: Command Line Basics Anthropic: Claude Code Overview

Block 4 — Reading⏱ 1 hr
📚 APIs & HTTP Fundamentals
Understand how software talks to software.
  1. What is an API (20 min): Endpoints, requests, responses, JSON format
  2. HTTP methods (15 min): GET, POST, PUT, DELETE — when to use each
  3. Status codes (10 min): 200, 201, 400, 401, 403, 404, 500 — what they mean
  4. Authentication (15 min): API keys, Bearer tokens, headers
Resources: MDN: HTTP Docs REST API Tutorial curl Manual
